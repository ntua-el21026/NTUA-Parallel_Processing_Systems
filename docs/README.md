# docs - Course Material and Reports

This folder collects PDFs and supporting material used across the assignments.

## Contents
- `reports/`: individual and final reports for assignments (PDF).
- `lecture_slides/`: course slides (PDF, Greek titles).
- `misc/`: lab guides and auxiliary references.
- `project_structure/`: helper script and text snapshot of the repository structure.

These documents are referenced by the per-assignment READMEs to provide context, methodology, and results.
